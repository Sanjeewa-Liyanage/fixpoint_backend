
# 🛠️ FixPoint Backend – After-Sale Service Management System

**FixPoint** is a backend system designed for hardware servicing companies that support banks. Built with PHP and PostgreSQL, it streamlines repair tracking, inventory management, Quality Checker operations, and technician scheduling with support for remote troubleshooting and route-based planning.

---

## 🚀 Key Features

- 🔁 Quarterly service scheduling and logs  
- 🔧 Repair workflow with backup unit tracking  
- 📦 Inventory management with alerts  
- 🛰️ Jitsi-based remote troubleshooting  
- 🗺️ Location-based branch clustering (PostGIS)  
- ✔️ Quality Checker verification tracking  
- 📊 Client-specific reporting  
- 🔐 Role-based access for Manager, Technician, and Quality Checker  

---

## ⚙️ Tech Stack

| Component    | Technology          |
|--------------|---------------------|
| Backend      | PHP   |
| Database     | PostgreSQL |
| APIs         | Google Places, Jitsi Meet |

---

## 🧪 Run Locally

### Prerequisites

- PHP 8.x
- PostgreSQL with PostGIS enabled

### Run Command

```bash
php -S localhost:8000
```

Then visit: [http://localhost:8000](http://localhost:8000)
